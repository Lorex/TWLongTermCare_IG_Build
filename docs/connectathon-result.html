聯測成果說明 (Summary)

各參測系統於本次聯測活動完成之驗測項目摘要如下：

  1. OAuth2 身份認證機制：所有參測系統皆通過 OAuth2 存取認證，具備符合 FHIR 標準的資料交換能力。
  2. 日間照護服務資料交換：5 個系統實作生理量測數據的上傳與查詢功能，其中 2 個系統同時支援用藥紀錄管理。
  3. 失智症照護監測：4 個系統實作失智症認知評估（MMSE/CDR）的資料交換功能。
  4. 機構間轉銜服務：2 個系統完整實作照護管理評估量表與 AA12 長期照護醫師意見書的交換機制。
  5. 長照 2.0 服務計畫與紀錄：本次無系統參與驗證。
