Resource Profile: 長期照顧－案件管理基礎

Official URL: http://ltc-ig.fhir.tw/StructureDefinition/LTCEpisodeOfCareBase Version: 1.0.0

此 Profile 為所有長期照顧 EpisodeOfCare Profile 的共用基礎，定義案件管理之基本結構與共用欄位。
