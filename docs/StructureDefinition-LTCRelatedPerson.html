Resource Profile: 長期照顧－關係人

Official URL: http://ltc-ig.fhir.tw/StructureDefinition/LTCRelatedPerson Version: 0.2.0
Active as of 2024-10-23 Computable Name: LTCRelatedPerson

此 Profile 說明本 IG 如何進一步定義 FHIR 的 RelatedPerson Resource，以呈現住民的關係人，包含家屬、照顧者、法定代理人等與住民有關的人員資料。

Usage:

  • Use this Resource Profile: 照顧管理評估量表文件打包
  • Refer to this Resource Profile: 長期照顧－照顧計畫, 長期照顧－照顧團隊, 照顧管理評估量表文件架構, 長期照顧－病情、問題或診斷... Show 3 more, 長期照顧－照顧目標, 長期照顧－問卷回覆 and 運動處方－運動計畫
