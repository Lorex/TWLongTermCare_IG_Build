Resource Profile: 長期照顧－服務人員角色

Official URL: http://ltc-ig.fhir.tw/StructureDefinition/LTCPractitionerRole Version: 0.2.0
Active as of 2024-10-23 Computable Name: LTCPractitionerRole

此 Profile 說明本 IG 如何進一步定義 FHIR 的 PractitionerRole Resource，以呈現長期照顧服務人員的角色資料。

有關 PractitionerRole 與 Practitioner 的差異及相關解釋，請參照 TW Core IG 中的說明。
