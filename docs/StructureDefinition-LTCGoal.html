Resource Profile: 長期照顧－照顧目標

Official URL: http://ltc-ig.fhir.tw/StructureDefinition/LTCGoal Version: 0.3.0
Active as of 2025-09-26 Computable Name: LTCGoal

此 Profile 說明本 IG 如何進一步定義 FHIR 的 Goal Resource，以呈現長期照顧情境中涉及之照顧目標內容。

Usages:

  • Use this Profile: 長期照顧管理中心個案服務初篩表/轉介單文件打包
  • Refer to this Profile: 長期照顧－照顧計畫
  • Examples for this Profile: Goal/ltc-goal-mobility-improvement-example
  • CapabilityStatements using this Profile: 臺灣長期照顧實作指引 - 用戶端能力聲明 and 臺灣長期照顧實作指引 - 伺服端能力聲明
