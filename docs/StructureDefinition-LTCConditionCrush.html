Resource Profile: 長期照顧－壓傷狀況

Official URL: http://ltc-ig.fhir.tw/StructureDefinition/LTCConditionCrush Version: 0.3.0
Active as of 2025-09-26 Computable Name: LTCConditionCrush

此 Profile 說明本 IG 如何進一步定義 FHIR 的 Condition Resource，以呈現長照機構住民的壓傷狀況。

Usages:

  • Use this Profile: 長期照顧管理中心個案服務初篩表/轉介單文件打包
  • Refer to this Profile: 長期照護管理中心個案服務初篩表/轉介單文件架構
  • Examples for this Profile: Condition/ltc-condition-crush-none-example and Condition/ltc-condition-crush-stage2-example
  • CapabilityStatements using this Profile: 臺灣長期照顧實作指引 - 用戶端能力聲明 and 臺灣長期照顧實作指引 - 伺服端能力聲明
